QuickTranslate

A free macOS menu bar app for instant translation.
Select text, press Ctrl+T, done.

What is QuickTranslate?

QuickTranslate sits in your menu bar and translates selected text from any app using a global keyboard shortcut. It uses the Claude API or OpenRouter for translation.

  • 18 languages — Korean, English, Japanese, Chinese, Spanish, French, German, Portuguese, Russian, Vietnamese, Thai, Indonesian, Italian, Dutch, Arabic, Hindi, Turkish
  • Global hotkey — Works in any app. Default: Ctrl+T, customizable
  • Multiple AI providers — Claude API or OpenRouter with 8+ models
  • Floating result panel — Original + translation, resizable, scrollable
  • Configurable language pair — Pick any two languages
  • Keychain storage — API keys never leave your machine
  • No tracking, no telemetry, no accounts

How to Use

  1. Install and open QuickTranslate. It appears in your menu bar.
  2. Click the menu bar icon → Settings → enter your API key (Anthropic or OpenRouter).
  3. Grant Accessibility permission (System Settings → Privacy & Security → Accessibility).
  4. Select text in any app, press Ctrl+T.
  5. Translation appears in a floating panel.

Requirements

  • macOS 14 (Sonoma) or later
  • Claude API key or OpenRouter API key
  • Accessibility permission
